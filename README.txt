Sentinel AI Dashboard
Name: Bhumika Battineni
Course: CIS 568

How to Run:
1. Unzip the folder.
2. Open 'index.html' in any modern web browser (Chrome/Firefox).
3. (Optional) For best performance, run via a local server (e.g., VS Code Live Server or 'python -m http.server').

Description:
A D3.js probabilistic simulation dashboard visualizing AI-driven cyber threats using force-directed graphs and geospatial projections.
